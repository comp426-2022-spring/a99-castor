Helpful Links

North Carolina COVID-19 Dashboard

This dashboard is managed by NCDHHS as apart of the COVID-19 response. It launched on March 23, 2022 and is updated every Wednesday. Data includes ER visits for COVID symptoms, COVID-19 particles found in water, weekly reported COVID-19 Cases, and percentage of NC population boosted.

Orange County COVID-19 Dashboard

Orange County's COVID-19 dashboard displays frequently updated charts that display statistics relating to COVID-19 in Orange County, North Carolina. There are details including cases by race, cases by ethnicity, weekly trends in Orange County, and a vaccine dashboard with vacination progress summary.

Carolina COVID-19 Dashboard

Although the Carolina COVID-19 Dashboard is no longer being frequently updated, it provides archived charts from earlier COVID-19 trends on the UNC campus that we encourage users to look through. It was lasted updated April 1, 2022; while still used it was updated weekly.

Gillings School of Public Health Data Visualization

UNC's Gillings School of Public Health provides an overview of COVID-19 cases by location in the state of North Carolina. users can interact with the chart and see the changes in trends over the past couple of months. It is updated every night at 2:30am, which is useful for tracking case trends more frequently.
